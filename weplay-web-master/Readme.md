
# weplay-web

## How to install

```bash
$ npm install
```

Then run it with the following ENV vars:

- `WEPLAY_PORT` - pointing to the port you want to listen on (`3000`)
- `WEPLAY_IO_URL` - io server url (`http://localhost:3001`)
- `WEPLAY_REDIS` - redis uri (`localhost:6379`)

```bash
$ node index
```
